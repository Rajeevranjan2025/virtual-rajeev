# Virtual Rajeev

Virtual Rajeev is a local AI assistant built as a learning project.

## Features
- Local LLM using Ollama
- Retrieval Augmented Generation (RAG)
- Personal knowledge base
- Privacy-aware responses
- Telegram bot integration

## Purpose
This project was built to understand how AI assistants work in practice,
focusing on system design, retrieval, and behavior rather than model training.

## Tech Stack
- Python
- Ollama
- FAISS
- Sentence Transformers
- Telegram Bot API
